Arena Tickets Widget — Attribute Matrix

Demonstrates all supported attributes and CSS custom properties for the <arena-tickets> custom element.

Supported Attributes

Attribute Type Default Description
feed-token string "" Public feed token identifying the event catalogue. Required to load data.
session-id string "" Optional UUID of a specific event session to deep-link into.
locale string (BCP-47) "en" Language/locale tag. Supported: en, ru, de, fr, es, uk.
CSS Variable Default Description
--arena-font-family system-ui Widget font family.
--arena-color-primary #1a1a1a Primary text colour.
--arena-color-secondary #6b7280 Secondary / muted text colour.
--arena-accent #6366f1 Accent / brand colour (buttons, links).
--arena-bg transparent Widget background colour.
--arena-radius 8px Border radius for cards/buttons.
--arena-border-color #e5e7eb Border / divider colour.

1 — feed-token only (minimum required)

<arena-tickets feed-token="demo-ft-001"></arena-tickets>

2 — feed-token + session-id

<arena-tickets
  feed-token="demo-ft-001"
  session-id="01930000-0000-7000-8000-000000000042"
></arena-tickets>

3 — locale: ru

<arena-tickets feed-token="demo-ft-002" locale="ru"></arena-tickets>

4 — locale: de

<arena-tickets feed-token="demo-ft-003" locale="de"></arena-tickets>
