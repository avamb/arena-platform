WHAT WAS DONE

Promoted the previously-private cfg.validate() to an exported method

to "aggregate every issue and return them all at once" using errors.Join
(stdlib, Go 1.20+). One boot attempt now reveals the full configuration
fault tree instead of one error at a time.
